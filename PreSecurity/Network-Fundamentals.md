# Network Fundamentals

## Overview

This module introduced the core concepts of computer networking, including how devices communicate, how data travels across networks, and the technologies that make modern communication possible.

### Topics Covered

## OSI Model

### Layer 1 - Physical
Hardware and transmission media.

### Layer 2 - Data Link
MAC addressing and frame delivery.

### Layer 3 - Network
IP addressing and routing.

### Layer 4 - Transport
TCP and UDP communication.

### Layer 5 - Session
Managing communication sessions.

### Layer 6 - Presentation
Data translation and encryption.

### Layer 7 - Application
User-facing network services and applications.

## TCP vs UDP

### TCP
- Reliable communication
- Error checking
- Connection-oriented

### UDP
- Faster communication
- No delivery guarantee
- Connectionless

## Ports

Common ports studied:
- 22 SSH
- 80 HTTP
- 443 HTTPS

### Topics Covered

- VLANs

## What I Learned

- Every device on a network requires unique identifiers.
- IP addresses and MAC addresses serve different purposes.
- Routers connect networks while switches connect devices.
- TCP prioritizes reliability while UDP prioritizes speed.
- The OSI model provides a framework for understanding network communication.
- Firewalls, VPNs, and subnetting improve security and network management.

## Real-World Application

Network fundamentals provide the foundation for security monitoring, log analysis, incident response, and SOC operations.
