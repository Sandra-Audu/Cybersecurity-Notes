# Network Fundamentals

## Key Concepts

- IP Addresses
- MAC Addresses
- DNS
- TCP and UDP
- Routers and Switches
- OSI Model
- Common Network Ports

## What I Learned

Network devices communicate using IP addresses.

DNS translates domain names into IP addresses.

TCP provides reliable communication while UDP prioritizes speed.

Routers connect different networks while switches connect devices within the same network.

## Important Ports

- 22 SSH
- 53 DNS
- 80 HTTP
- 443 HTTPS

## Real World Application

Network fundamentals are essential for monitoring traffic, analyzing logs, and investigating security incidents as a SOC Analyst.
